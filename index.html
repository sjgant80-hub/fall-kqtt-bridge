How to integrate in GoldenShower (one line)

In docs/golden-shower/multiplayer/lobby.md right after network.init(code):

// ◊ optional · join the AI Native Solutions mesh
if (typeof GS.FallKQTTBridge !== 'undefined') {
  GS.fallBridge = GS.FallKQTTBridge.attach(network.kqtt, code);
}

Bridge module is 50 LOC. Add a <script src="fall-kqtt-bridge.js"> tag (or paste the code block into a markdown file the markdown-runner picks up). Zero new transport, zero dependencies, opt-in toggles.
